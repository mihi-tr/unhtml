unhtml - escape HTML easily

What?

This is a simple HTML escaper: pass a query string to an API and get a string - without the HTML - back.

I created this, since converting HTML to text - especially with HTML entities is a slight pain. It might be easier to use a WEB API. Also currently removing HTML tags relies on regexes - not the best way to do it. And finally - writing a google appengine app for fun and profit.

API

curl "http://unhtml.appspot.com/escape?<p>Mot&ouml;rhead</p>"

will return the string 

Motörhead

Use it in Refine

To use it in Refine select "Add column by fetchin urls" and use the following expression: 

"http://unhtml.appspot.com/escape?"+value.replace("#","%23")
